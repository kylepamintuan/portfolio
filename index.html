Profile

Hi, I'm Kyle. I'm a web developer & programmer based in Los Angeles, CA.

Growing up I've always taken interest in art, design and things relating to problem-solving. Thus, after taking my first programming course in 2013 I knew that designing and building apps were things that I naturally enjoyed doing and so I decided at that point to continue down that path. Since then I've gained a lot of theoretical and practical knowledge of programming such as the syntax/semantics of various programming languages, what goes on behind the code, data structures, algorithms, agile methodology, version control, debugging, full- stack web development and so on.

Now in 2018 I'm proud to say that I've completed a CS degree program at CSULB as well as a web development program at LearningFuze. I believe the foundational programming knowledge I learned from my CS degree combined with the development tools and techniques I acquired from my coding bootcamp has prepared me well for shaping the future of web development.

Let's collaborate and build something great.
